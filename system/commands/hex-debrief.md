# /hex-debrief — Weekly Debrief

Walk through projects, priorities, and progress. Best run on Fridays or Mondays.

## Steps

1. **Projects review:** For each directory in `projects/` (excluding `_archive/`):
   - Read `context.md` — what's the current status?
   - Any decisions made this week? Check `decisions/` for recent entries.
   - Any blockers? Surface them.

2. **Priorities review:** Read `todo.md`:
   - What moved from "Now" to "Completed" this week?
   - What's been in "Now" for more than 2 weeks without progress? Flag as stalled.
   - What's in "Waiting On Others" that needs a nudge?

3. **People review:** Check `people/` for any profiles updated this week. Surface relationship context that might be relevant.

4. **Evolution review:** Read `evolution/changelog.md` for improvements made this week. Read `evolution/suggestions.md` for pending proposals.

5. **Propose next week:** Based on the review, suggest 3-5 priority items for next week. Ask the user to confirm or adjust.

6. **Report:** Summary of the week — what shipped, what's stuck, what's next.
