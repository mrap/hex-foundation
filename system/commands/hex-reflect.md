# /hex-reflect — Session Reflection

Review the current session to extract learnings and identify failures.

## Steps

1. **Scan for corrections:** Did the user correct the agent during this session? For each correction:
   - What was the error?
   - What should have happened?
   - Is there already a standing order covering this? If yes, note the violation. If no, propose one.

2. **Scan for friction:** Did the user get stuck, express frustration, or repeat themselves? Record each friction point.

3. **Scan for patterns:** Did the user demonstrate a preference not yet recorded in `me/learnings.md`? Record it.

4. **Update learnings:** Write new observations to `me/learnings.md` with evidence and today's date.

5. **Update evolution:** Write new friction patterns to `evolution/observations.md`. If any pattern now has frequency >= 3, generate a suggestion in `evolution/suggestions.md`.

6. **Report:** "Reflection complete. N learnings recorded, M friction patterns logged."
